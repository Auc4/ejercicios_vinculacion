# Documentación Interna - Clase 4

Este documento detalla las modificaciones y adiciones realizadas en los archivos de la Clase 4, sirviendo como referencia para tutores y alumnos que deseen revisar los cambios entre la versión inicial (código de la Clase 3) y la versión final de esta lección.

##  Cambios Clave en la Lección

La Clase 4 se centró en (Ej. **Enlaces, imágenes y atributos de accesibilidad**). Los principales cambios se realizaron en la adición y configuración de elementos multimedia e interactivos en el código HTML.

##  Detalle de Modificaciones

A continuación, se presenta una lista comparativa de las adiciones y ajustes realizados en el código.

### 1. HTML (`index.html` vs `resultado.html`)

| Descripción del Cambio | Propósito / Etiqueta Afectada |
| :--- | :--- |
| **[Cambio 1: Ej. Inserción de una imagen de perfil]** | Se añadió la etiqueta `<img>` y se configuraron los atributos `src` y `alt` para insertar la foto del perfil. |
| **[Cambio 2: Ej. Adición de un enlace externo]** | Se incorporó la etiqueta `<a>` dentro del `<footer>` para enlazar a la red social X (anteriormente Twitter), con el atributo `target="_blank"`. |
| **[Cambio 3: Ej. Creación de un botón de acción]** | Se añadió un enlace con la clase `btn-trabajo` para simular un botón de acción en la sección de "Mi Trabajo". |
| **[Cambio 4: Ej. Organización de elementos en un contenedor]** | Se agruparon varios elementos (imagen, texto) dentro de un `<div>` con la clase `contenedor-flexible`. |
