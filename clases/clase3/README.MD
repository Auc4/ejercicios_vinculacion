# Documentación Interna - Clase 3

Este documento detalla las modificaciones y adiciones realizadas en los archivos de la Clase 3, sirviendo como referencia para tutores y alumnos que deseen revisar los cambios entre la versión inicial (`index.html`) y la versión final (`resultado.html`).

##  Cambios Clave en la Lección

La Clase 3 se centró en (Ej. **Introducción a las etiquetas semánticas y estructura básica de un documento HTML5**). Los principales cambios se realizaron en la estructura del código HTML, mejorando la organización y accesibilidad del documento.

##  Detalle de Modificaciones

A continuación, se presenta una lista comparativa de las adiciones y ajustes realizados en el código.

### 1. HTML (`index.html` vs `resultado.html`)

| Descripción del Cambio | Propósito / Etiqueta Afectada |
| :--- | :--- |
| **[Cambio 1: Ej. Reemplazo de un `<div>` genérico]** | Se sustituyó un `<div>` por la etiqueta `<header>` para definir la sección de encabezado principal del sitio. |
| **[Cambio 2: Ej. Adición de una nueva sección]** | Se añadió la etiqueta `<section>` para agrupar el contenido principal de la página, mejorando la estructura semántica. |
| **[Cambio 3: Ej. Actualización de enlaces]** | Se modificaron los atributos `href` de los enlaces de navegación dentro del `<nav>`. |
| **[Cambio 4: Ej. Creación del pie de página]** | Se incorporó la etiqueta `<footer>` para contener información de derechos de autor y enlaces secundarios. |
