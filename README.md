# Proyecto de Vinculación: Ejercicios de HTML y CSS

Bienvenidos al repositorio de ejercicios para el proyecto de vinculación. Este documento les va a servir como guía para que dicten sus respectivas clases.

- Contenido del Repositorio

Todo el material de enseñanza se encuentra centralizado en la carpeta principal llamada `/clases`.
Dentro de esta carpeta, encontrarán el material dividido en subcarpetas numeradas que corresponden a cada sesión o lección (por ejemplo: `clase1`, `clase2`, etc.).

- Dinámica de las Clases

Cada carpeta de clase (ej. `/clase1`) sigue una estructura estándar diseñada para facilitar la enseñanza:

* **index.html:** Este es el archivo de inicio. Es el documento base que se debe utilizar para comenzar la lección, mostrar los ejemplos y codificar en vivo junto a los estudiantes.
* **resultado.html:** Este es el resultado esperado. Funciona como una "clave de respuestas" que muestra cómo debería quedar el código al finalizar la clase. Sirve como guía de comparación para tutores y estudiantes.
* **Anexos (Carpetas adicionales):** Además de los archivos HTML, es posible que encuentren carpetas de apoyo como `/imagenes` o archivos `.css`, los cuales son necesarios para completar los ejercicios de esa lección.
* **Documentación Interna:** En la carpeta de clases hay un **.MD** explicando detalladamente los **cambios entre los archivos** (`index.html` vs. `resultado.html` y el código CSS).
